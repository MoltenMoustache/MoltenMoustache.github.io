Molten's Behaviour Tree Creator

Molten's BT Creator is an asset package I created for Unity, for use in future projects. The assets facilitiates the easy development of complex A.I behaviour through custom procedural Behaviour Trees.

The package currently contains the ability to create a BehaviourTree object, as well as composite, leaf and decorator nodes, which when combined can result in fairly complex A.I behaviour.

Contribution

  • Implemented Composite nodes in the form of Selectors and Sequences
  • Implemented Custom Leaf Nodes
  • Implemented Decorator nodes in the form of Repeater, Retry, Inverter, Succeeder and Failer
  • Created a small demo scene, demonstrating an A.I Agent interupting his movement path to chase the player
